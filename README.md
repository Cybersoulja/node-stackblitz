# Jekyll Site

This is a Jekyll site deployed with GitHub Pages.

## Local Development

1. Install dependencies: `bundle install`
2. Run the site locally: `bundle exec jekyll serve`
3. Open your browser at `http://localhost:4000`

## Deployment

The site automatically deploys to GitHub Pages on pushes to the `main` branch via the workflow in `.github/workflows/jekyll-gh-pages.yml`.
